Float Property

PictureElement
Apr 07, 2017

Summary

With float you can place block elements along the left or right side of their container and make inline elements (and text) to wrap around it.

Examples

  1. The image is float to the left and the rest of the content is wrap around it.

    HTML
    <div>
      <p>Paragraph 1</p>
      <p>Paragraph 2</p>
      <p>Paragraph 3</p>
      <img src="images/avatar.png"alt="avatar" width="100">
      <p>Paragraph 4</p>
      <p>Paragraph 5</p>
      <p>Paragraph 6</p>
      <p>Paragraph 7</p>
      <p>Paragraph 8</p>
      <p>Paragraph 9</p>
      <p>Paragraph 10</p>
      <p>Paragraph 11</p>
      <p>Paragraph 12</p>
    </div>

  2. Floated elements (in this case <li> elements) stack up to their parent's edge. If the parent's width becomes too small to contain all of the list items, the third list item is going to break down to a new line.

    HTML

      <li><img src="images/avatar.png"alt="avatar" width="100"></li>
      <li><img src="images/avatar.png"alt="avatar" width="100"></li>
    </ul>
    • CSS
      ul {
        width: 500px;
        height: 250px;

        list-style-type: none;

  3. Take care with elements that have different heights - the first available edge isn't always exactly below.

    HTML
    <ul>
      <li><img src="images/avatar.png"alt="avatar" width="100"></li>
      <li><img src="images/avatar.png"alt="avatar" width="100"></li>
      <li><img src="images/avatar.png"alt="avatar" width="250"></li>
    </ul>

    ul {

      height: 350px;

      list-style-type: none;
    }

    li {

Clearing Floats

Description

An element that has the clear property set on it, will not move adjacent to the float like the float desires, but will move itself down past the float.

Syntax

CSS
clear: left|right|both;

where:

  • left: clear any floats coming from the left.
  • right: clear any floats coming from the right.
  • both: clear any floats coming from both (left and right) sides.

Examples

  1. HTML
    <div>
      <img id="float-image-1" src="images/avatar.png"alt="avatar" width="150">
      <img id="float-image-2" src="images/avatar.png"alt="avatar" width="100">
      <p>
        The quick brown fox jumps over the lazy dog.
        The quick brown fox jumps over the lazy dog.
        The quick brown fox jumps over the lazy dog.
        The quick brown fox jumps over the lazy dog.
        The quick brown fox jumps over the lazy dog.
        The quick brown fox jumps over the lazy dog.
        The quick brown fox jumps over the lazy dog.
        The quick brown fox jumps over the lazy dog.
      </p>
    </div>
    CSS
    div {
      padding: 0;
      margin: 0;
      width: 500px;
      height: 250px;
      border: 5px solid green;
    }

    #float-image-1 {
      float: left;
    }

    #float-image-2 {
      float: right;
    }

    • The <p> element has no clear property set on it. The element will move adjacent to the floats like the floats desire.
