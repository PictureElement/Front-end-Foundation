Float Property

PictureElement
Apr 07, 2017

Summary

With float you can place block elements along the left or right side of their container and make inline elements(and text) to wrap around it.

Example

HTML
<div>
  <p>Paragraph 1</p>
  <p>Paragraph 2</p>
  <p>Paragraph 3</p>
  <img src="images/avatar.png"alt="avatar" width="100">
  <p>Paragraph 4</p>
  <p>Paragraph 5</p>
  <p>Paragraph 6</p>
</div>

a)

CSS
div {
  border: 5px solid green;
}
