Box Shadow

Description

The box-shadow property enables you to cast one or more shadows from the frame of almost any element.

Syntax

box-shadow: inset offset-x offset-y blur-radius spread-radius color

where:

  • inset (optional): if not specified (default), a drop shadow is created rather than an inset shadow.
  • offset-x: moves the shadow along the x-axis (left & right).
  • offset-y: moves the shadow along the y-axis (up & down).
  • blur-radius (optional): changes the blur amount of the shadow. The larger the value, the bigger the blur, so the shadow becomes bigger and lighter. Negative values are not allowed. If not specified the value will be 0 (shadow's edge is sharp).
  • spread-radius (optional): causes the shadow to expand (grow bigger) or shrink. In other words, it changes the size of the shadow.
  • color (optional): specifies the color of the shadow.

Examples

  1. Sharp shadow edge (no blur):

    box-shadow: 5px 10px black;

  2. Sharp shadow edge (no blur):

    box-shadow: -5px 10px black;

  3. Sharp shadow edge (no blur):

    box-shadow: 5px -10px black;

  4. Sharp shadow edge (no blur):

    box-shadow: -5px -10px black;

  5. Specify a blur radius:

    box-shadow: 5px 10px 5px black;

  6. Specify a spread radius (expand shadow):

    box-shadow: 5px 10px 5px 5px black;

  7. Specify a spread radius (shrink shadow):

    box-shadow: 5px 10px 5px -5px black;

  8. Specify a spread radius instead of blur-radius:

    box-shadow: 5px 10px 0 5px black;

  9. Inset and outset/drop shadow combination:

    box-shadow:
      inset 5px 10px 5px gray,
      5px 10px 5px black;

  10. Specify multiple box shadows:

    box-shadow:
      0 0 10px black,
      /* x: right, y: down */
      10px 5px yellow,
      /* x: left, y: down */
      -10px 5px red,
      /* x: left, y: up */
      -10px -5px blue,
      /* x: right, y: up */
      10px -5px lime;

Text Shadow

Description

The text-shadow property adds shadows to text.

Syntax

text-shadow: offset-x offset-y blur-radius color
/* OR */
text-shadow: color offset-x offset-y blur-radius

where:

  • offset-x: position of horizontal shadow.
  • offset-y: position of vertical shadow.
  • blur-radius (optional): controls the blur amount. The higher this value, the bigger the blur and the shadow becomes wider and lighter.
  • color (optional): specifies the color of the shadow. This value can be specified either before or after the offset values.

Note: You can apply multiple text shadows, using a comma-separated list.

Multiple Backgrounds

Description

CSS3 allows you to apply multiple backgrounds to an element. Backgrounds are stacked on top of each other. The first background you provide is on top of the stack and the last background listed in the back.

Syntax

  1. Specify individual properties:

    CSS

    .myclass {
      background-image: value1, ..., valueN;
      background-position: value1, ..., valueN;
      background-size: value1, ..., valueN;
      background-repeat: value1, ..., valueN;
    }
  2. Shorthand:

    CSS

    .myclass {
       background:
        bg-image1 bg-position1 bg-size1 bg-repeat1 bg-color1,
        ...
        bg-imageN bg-positionN/bg-sizeN bg-repeatN bg-colorN;
    }

    Note: If one of the properties in the shorthand syntax is the bg-size property, you must use a slash(/) to separate it from the bg-position property (e.g. 10px 20px/50px 40px, top/cover, etc.)

Example

HTML

<div class="mult-bg-1">
</div>

CSS

.mult-bg-1 {
  border: 5px solid black;
  height: 300px;
  background-image:
    url(images/avatar.png),
    url(images/lightning.svg),
    linear-gradient(to left, #dc2424, #4a569d);
  background-position: 50% 20%, center, center;
  background-size: 150px, contain, contain;
  background-repeat: no-repeat, no-repeat, no-repeat;
}
/* OR (Shorthand) */
.mult-bg-1 {
  border: 5px solid black;
  height: 300px;
  background:
    url(images/avatar.png) 50% 20%/150px no-repeat,
    url(images/lightning.svg) center/contain no-repeat,
    linear-gradient(to left, #dc2424, #4a569d) center/contain no-repeat;
}

Result

Gradients

Description

Gradients let us create smooth transitions between two or more specified colors. There are two types of gradients: linear gradients and radial gradients.

Linear gradients

A linear gradient is a straight/linear progression.

CSS

background: linear-gradient(direction, color-stop1, ..., color-stopN);

where:

  • direction describes the direction of the gradient and can be specified by a side-or-corner or an angle.
  • color-stop is a point along the gradient line that has a specific color at that location. A color-stop value is comprised of a color value, followed by an optional stop position.

where:

  • side-or-corner is optional and equals to:
    • to right (gradients runs from left to right)
    • to left (gradients runs from right to left)
    • to top (gradients runs from bottom to top)
    • to bottom (gradients runs from top to bottom)
    • to top left (gradient runs diagonally)
    • to top right (gradient runs diagonally)
    • to bottom left (gradient runs diagonally)
    • to bottom right (gradient runs diagonally)
    Note: The order is not relevant(e.g. to top left = to left top)
  • angle is optional and is specified as an angle between a vertical line and the gradient line. Some examples are:
    • 90deg (similar to: to right)
    • -90deg (similar to: to left)
    • 0 (similar to: to top)
    • 180deg (similar to: to bottom)
  • color can be specified in any of the following ways:
    • #RRGGBB
    • #RGB
    • #RRGGBBaa
    • #RGBa
    • rgb(r,g,b)
    • rgb(pctr,pctg,pctb)
    • rgba(r,g,b,a)
    • hsl(hue,saturation,lightness)
    • hsla(hue,saturation,lightness,a)
  • stop position is optional and can be specified as either a percentage, or as a length. For example, the starting point is represented with 0% (or a length of 0) and the ending point with 100%.

  <div class="first">

  </div>

Radial gradients

A radial gradient is a circular or elliptic progression. This type of gradient, creates a gradient that extends from an origin (i.e.the center of the element), extending outward in a circular or elliptical pattern.

CSS

background: radial-gradient(<shape> <size> <at position>, color-stop1, ..., color-stopN);

where:

  • shape is optional, defines the shape of the gradient, and equals to:
    • circle
    • ellipse (default)
  • size is optional, defines the size of the ending shape, and equals to:
    • closest-side
    • farthest-side
    • closest-corner
    • farthest-corner (default)
  • position is optional, defines the position of the gradient, and equals to:
    • left
    • center (default)
    • right
    • top
    • bottom
    • left top
    • left center
    • left bottom
    • center top
    • center center
    • center bottom
    • right top
    • right center
    • right bottom
    • top left
    • top center
    • top right
    • bottom left
    • bottom center
    • bottom right
    Note: left, center, right, top, and bottom are identical to left center, center center, right center, top center, and bottom center respectively.
  • color-stop is a point along the gradient line that has a specific color at that location. A color-stop value is comprised of a color value, followed by an optional stop position

where:

  • color can be specified in any of the following ways:
    • #RRGGBB
    • #RGB
    • #RRGGBBaa
    • #RGBa
    • rgb(r,g,b)
    • rgb(pctr,pctg,pctb)
    • rgba(r,g,b,a)
    • hsl(hue,saturation,lightness)
    • hsla(hue,saturation,lightness,a)
  • stop position is optional and can be specified as either a percentage, or as a length. For example, the center of the gradient is represented with 0% (or a length of 0).

Example
