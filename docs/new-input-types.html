Description

<input type="search"> is a one-line plain-text for entering one or more search terms.

Example

HTML5

Number

Description

<input type="number"> represents a precise control for setting the element's value to a string representing a number. In the browser we get a convinient control to the right side of the input that allows us to increase or decrease the value. The keyboard on mobile devices changes to have a row of number keys at the top.

Example

HTML5

<form>
  Enter a number (between 1 and 5):
  <input type="number" name="number" min="1" max="5">
</form>

Result

Enter a number (between 1 and 5):

Range

Description

Example

HTML5
