Selectors

PictureElement
Mar 14, 2017

Type Selector

Description

This selector directly matches one or more elements based on the type of the element. A type selector for a particular node name selects all elements of that type.

Example

CSS

p {
  color: black;
}

Attribute Selector

Description

This type of selector matches elements based on their attributes and attributes values.

Example

HTML

<ul>
  <li data-taxonomy-class>Arachnida</li>
  <li>Araneae</li>
  <li>Scorpiones</li>
  <li>Theraphosidae</li>
  <li>Buthidae</li>
  <li>Brachypelma</li>
  <li>Androctonus</li>
  <li data-taxonomy-species="Mexican terrestrial redknee">B. smithi</li>
  <li data-taxonomy-species="Fat-tailed potent man-killer">A. australis</li>
</ul>

Note: There is a defined set of attributes for each HTML element. You may choose to omit some but you cannot choose to add further attributes beyond those defined, unless you consider using a HTML5 custom data attribute:

<element data-attributename="value">
  ...
</element>

CSS

[data-taxonomy-class] {
  color: blue;
}

[data-taxonomy-species="redknee"] {
  background-color: green;
}

[data-taxonomy-species~="man-killer"] {
  background-color: red;
}

Result

  • Arachnida
  • Araneae
  • Scorpiones
  • Theraphosidae
  • Buthidae
  • Brachypelma
  • Androctonus
  • B. smithi
  • A. australis

Multiple Selector

Description

This type of selector is used to apply the same rule to multiple sets of selected elements at once.

Example

CSS

p, a {
  font-size: 12px;
  font-family: Lato, Helvetica, Arial;
}

h1, h1, h2 {
  font-family: Arvo, Times, "Times New Roman";
}

A B Combinator (a.k.a Descendant Selector)

Description

This type of selector, selects any element matching B that is inside of an element matching A.

Example

CSS

ul li {
  font-size: 24px;
}

This type of selector matches all <li> elements inside <ul> elements.

A + B Combinator (a.k.a Adjacent Sibling Selector)

Description

This type of selector, selects only the element matching B that immediately follows elements matching A.

Examples

  1. HTML

    <ul>
      <li>First child</li>
      <li>Second child</li>
      <li>Third child</li>
    </ul>

    CSS

    li:first-of-type + li {
      color: red;
    }

    Result

    • First child
    • Second child
    • Third child

  2. HTML

    <div>
      <p>paragraph 1</p>
      <p>paragraph 2</p>
    </div>
    <p>paragraph 3</p>
    <p>paragraph 4</p>

    CSS

    div + p {
      color: red;
    }

    Result

    paragraph 1

    paragraph 2

    paragraph 3

    paragraph 4

A > B Combinator (a.k.a Child Selector)

Description

This type of selector, selects all elements matching B that are immediate children of elements matching A.

Example

HTML

<div>
  <p>paragraph 1</p>
  <p>paragraph 2</p>
</div>
<p>paragraph 3</p>
<p>paragraph 4</p>

CSS

div > p {
  color: red;
}

Result

paragraph 1

paragraph 2

paragraph 3

paragraph 4

A ~ B Combinator (a.k.a General Sibling Selector)

Description

This type of selector, selects all elements matching B that are siblings of elements matching A.

Example

HTML

<div>
  <p>paragraph 1</p>
  <p>paragraph 2</p>
</div>
<p>paragraph 3</p>
<p>paragraph 4</p>

CSS

div ~ p {
  color: red;
}

Result

paragraph 1

paragraph 2

paragraph 3

paragraph 4

Pseudo-Class

Description

A pseudo-class is a keyword that can be added to selectors to match an element in a certain state.

Examples

  1. The :hover pseudo-class is used. The rule is only going to be applied when the mouse hovers over the link:

    HTML

    <a href="mailto:pictureelement@yandex.com">PictureElement</a>

    CSS

    a {
      text-decoration: none;
    }

    a:hover {
      text-decoration: underline;
      color: red;
    }

    Result

    PictureElement

  2. The :first-child pseudo-class is applied to narrow down the amount of child tags selected. The style is applied only on the first list item.

    HTML

    <ul>
      <li>First item</li>
      <li>Second item</li>
      <li>Last item</li>
    </ul>

    CSS

    ul li:first-child {
      color: red;
    }

    Result

    • First item
    • Second item
    • Last item

  3. The :last-child pseudo-class is applied to narrow down the amount of child tags selected. The style is applied only on the last list item.

    HTML

    <ul>
      <li>First item</li>
      <li>Second item</li>
      <li>Last item</li>
    </ul>

    CSS

    ul li:last-child {
      color: blue;
    }

    Result

    • First item
    • Second item
    • Last item

  4. The :nth-child and :nth-of-type pseudo-classes are applied to narrow down the amount of child tags selected.

    • In this example two styles are applied, red color on even list items and blue color on odd list items.

      HTML

      <ul>
        <li>First item</li>
        <li>Second item</li>
        <li>Third item</li>
        <li>Fourth item</li>
      </ul>

      CSS

      ul li:nth-child(even) {
        color: red;
      }

      ul li:nth-child(odd) {
        color: blue;
      }

      Result

      • First item
      • Second item
      • Third item
      • Fourth item

    • In this example, red color is applied on every third list item starting with third list item.

      HTML

      <ul>
        <li>First item</li>
        <li>Second item</li>
        <li>Third item</li>
        <li>Fourth item</li>
        <li>Fifth item</li>
        <li>Sixth item</li>
      </ul>

      CSS

      ul li:nth-child(3n) {
        color: red;
      }

      Result

      • First item
      • Second item
      • Third item
      • Fourth item
      • Fifth item
      • Sixth item

    • :nth-child vs :nth-of-type

      HTML

      <div>
        <h4>Heading 1</h4>
        <p>Paragraph 1</p>
        <p>Paragraph 2</p>
      </div>

      CSS

      div p:nth-child(2) {
        color: red;
      }

      div p:nth-of-type(2) {
        color: blue;
      }

      Result

      Heading 1

      Paragraph 1

      Paragraph 2

Note: There are more pseudo-classes available and you can learn more here.

Pseudo-Elements

Description

A pseudo-element is a keyword that can be added to selectors and allows you to style certain parts of an element.

Examples

  1. The ::first-letter pseudo-element matches the first letter of the first line of an element.

    HTML

    <p>
      Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut
      labore et dolore
    </p>

    CSS

    p::first-letter {
      font-size: 25px;
      color: red;
    }

    Result

    Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore

  2. The ::first-line pseudo-element matches the first line of an element.

    HTML

    <p>
      Line 1<br>
      Line 2<br>
      Line 3
    </p>

    CSS

    p::first-line {
      background-color: yellow;
    }

    Result

    Line 1
    Line 2
    Line 3

  3. The ::after/::before pseudo-elements insert something after/before the content of the selected element.

    HTML

    <p id="product">Product symbol:</p>
    <p id="sum">: Sum symbol</p>

    CSS

    #product::after {
      content: " \220F";
    }

    #sum::before {
      content: "\2211";
    }

    Result

    Product symbol:

    : Sum symbol

  4. The ::after/::before pseudo-elements insert something after/before the content of the selected element. In this example we'll add delimiters between navigation items, using the ::after/::before pseudo-elements.

    HTML

    <nav id="nav-delimiter">
      <button class="dropbtn">Item#1</button>
      <button class="dropbtn">Item#2</button>
      <button class="dropbtn">Item#3</button>
      <button class="dropbtn">Item#4</button>
    </nav>

    CSS

    #nav-delimiter button {
      padding: 14px 16px;
    }

    /*
      Select all <button> elements except the first and add a "|" delimiter before the content of   the selected elements.
    */
    #nav-delimiter button:not(:first-child)::before {
      content: "|";
      position: relative;
      left: -15px;
    }

    Result

ID selector

Description

An ID selector matches the element with the specified id.

Syntax

CSS

#id {
  properties
}

Example

HTML

<p>
  This is a basic example of ID selectors.
  <span id="highlight">This sentence is highlighted.</span>
<p>

CSS

#highlight {
  background-color: yellow;
}

/* This is also equivalent to the following attribute selector: */

[id="highlight"] {
  background-color: yellow;
}

Result

This is a basic example of ID selectors. This sentence is highlighted.

ID's and Classes

  • Each element can have only one ID, unique to the page.
  • If a style needs to be applied to only one element, an id selector should be used.
  • Each element can have multiple classes and the same class can be used on multiple elements.
  • If a style needs to be applied to multiple elements, a class selector should be used.

Class selector

Description

The .class selector selects elements with a specific class attribute.

Syntax

CSS

.class {
  declarations;
}

Examples

  1. Select and style only one <ul>:

    HTML

    <ul class="nav">
      <li><a href="home.html">Home</a></li>
      <li><a href="recipes.html">Recipes</a></li>
      <li><a href="suggest.html">Suggest</a></li>
    </ul>

    CSS

    .nav {
      declarations;
    }

  2. Change the style of only <a> tags that are inside of a <ul> tag with a class of nav:

    HTML

    <ul class="nav">
      <li><a href="home.html">Home</a></li>
      <li><a href="recipes.html">Recipes</a></li>
      <li><a href="suggest.html">Suggest</a></li>
    </ul>

    CSS

    .nav a {
      declarations;
    }

  3. We want only the navigation list to display inline:

    HTML

    <ul class="nav">
      <li><a href="home.html">Home</a></li>
      <li><a href="recipes.html">Recipes</a></li>
      <li><a href="suggest.html">Suggest</a></li>
    </ul>

    <ul>
      <li>2 eggs</li>
      <li>1 pound of brown sugar</li>
      <li>3 sticks of butter</li>
    </ul>

    CSS

    /*
    Select <li> elements that are children of <ul> elements with a class of nav and make them display inline.
    */
    .nav li {
      display: inline;
    }

  4. We need a way to access the creator paragraph tag in our page in order to add some specific properties. To achieve that we have to add a class called creator to each paragraph tag that contains the creator name:

    HTML

    <p class="creator">creator: Marios Sofokleous</p>

    CSS

    .creator {
      declarations;
    }

ID's and Classes

  • Each element can have only one ID, unique to the page.
  • If a style needs to be applied to only one element, an id selector should be used.
  • Each element can have multiple classes and the same class can be used on multiple elements.
  • If a style needs to be applied to multiple elements, a class selector should be used.

Cascade order

  1. HTML

    element {
      color: blue;
    }

    element {
      color: red;
    }

    The second color definition(red) overrides the first(blue).

  2. HTML

    element {
      color: cyan;
    }

    element {
      font-size: 20px;
    }

    Non-conflicting properties will be combined. The code section above is equivalent to the one below:

    HTML

    element {
      color: cyan;
      font-size: 20px;
    }

Inheritance

Description

Some nested elements inherit parent styles by default others aren't. However, inheritance is controllable. Which properties are inherited by default and which aren't is down to common sense and to be sure you can check the CSS reference.

Inherited properties:

  • color
  • font-family
  • font-related properties
  • text-align
  • text-shadow

Uninherited properties:

  • background
  • background-related properties
  • border
  • border-related properties
  • box-sizing
  • box-shadow
  • height
  • width
  • margin
  • padding
  • opacity

Examples

  1. HTML

    <div>
      <p>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt
        ut labore et dolore magna aliqua.
      </p>
    </div>

    a) The <p> element inherits the text-shadow property from its parent (<div> element). This is the default behaviour:

    CSS

    div {
      text-shadow: 0 1px 1px black;
    }

    Result

    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

    b) The <p> element inherits the text shadow property defined in the browser's default style sheet:

    CSS

    div {
      text-shadow: 0 1px 1px black;
    }

    /*
      The text-shadow property is inherited by default. We need a way to control inheritance. The
      initial keyword sets the property value applied to a selected element to be the same as the
      value set for that element in the browser's default style sheet.
    */
     div p {
      text-shadow: initial;
    }

    Result

    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

  2. HTML

    <div id="inheritance">
      <p>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut
        labore et dolore magna aliqua.
      </p>
      <div>
        <p>
          Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt
          ut labore et dolore magna aliqua.
        </p>
      </div>
    </div>

    a) The innner <div> element does not inherit the border property from its parent (outer <div> element). This is the default behaviour:

    CSS

    #inheritance {
      border: 1px solid red;
    }

    Result

    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

    b) The innner <div> element inherits the border property from its parent (outer <div> element):

    CSS

    #inheritance {
      border: 1px solid red;
    }

    /*
      The border property is not inherited by default. We need a way to control inheritance. The
      inherit keyword sets the property value applied to a selected element to be the same as the
      value set for its parent element.
    */
     #inheritance div {
      border: inherit;
    }

    Result

    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

Specificity

Description

Specificity refers to the priority of a particular selector.

Calculation

Specificity is a weight that is applied to a given CSS declaration. The following list of selector types increases in specificity:

  1. Type selectors and Pseudo-Elements
  2. Class selectors, Attribute selectors, and Pseudo-Classes
  3. ID selectors
  4. Inline styles

Note: Universal selector(*), Combinators(+, >, ~, '') and Negation Pseudo-Class(:not()) have no effect on specificity.

Examples

  1. p { ... } - Weight: 0,0,0,1
  2. .class { ... } - Weight: 0,0,1,0
  3. #id { ... } - Weight: 0,1,0,0
  4. <p style="color: red;">content</p> - Weight: 1,0,0,0
  5. ul#nav li:active a { ... } - Weight: 0,1,1,3
  6. ul > li ul li ol li:first-letter { ... } - Weight: 0,0,0,7
  7. p { color: red !important; } - Weight: Highest priority

    Note:The !important rule on a style declaration, overrides any other declarations (even inline styles). The only way that can be overridden is with another !important rule declared later in CSS.

  8. div:not(.outer) p { ... } - Weight: 0,0,1,2

    Note: The negation pseudo-class :not is not taken into consideration in the specificity calculation. However, any selectors placed into it are.

  9. HTML

    <div id="content" class="featured">
      <p>Break out the cocoa.<p>
    </div>

    CSS

    /* Weight: 0,1,0,0 */
    #content {
      color: #000;
    }

    /* Weight: 0,0,1,0 */
    .featured {
      color: #fff;
    }

    The first style declaration is the one that wins. The paragraph's font color is set to black.

  10. Give a black color to the section's content and a red color to Paragraph#1:

    HTML

    <section id="content">
      <p class="featured">Paragraph #1.</p>
      <p>Paragraph #2.</p>
      <p>Paragraph #3.</p>
    </section>

    • CSS (Wrong way)

      #content p { color: black; } - Weight: 0,1,0,1

      .featured { color: red; } - Weight: 0,0,1,0

    • CSS (Method #1)

      #content p { color: black; } - Weight: 0,1,0,1

      .featured { color: red !important; } - Weight: Highest priority

      Note:Always look for a way to use specificity before even considering !important.

    • CSS (Method #2)

      #content p { color: black; } - Weight: 0,1,0,1

      #content .featured { color: red; } - Weight: 0,1,1,0
