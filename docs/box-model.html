Box Model

PictureElement
Mar 15, 2017

Description

Every tag shown in the body is contained in an invisible rectangle called the box. The box model is a way to describe the borders and spacing in between the boxes of each tag. The box model allows us to add a border around elements, and to define space between elements.

Properties

  1. Content area: actual content of the box, where text and images appear.
  2. Padding: added to the top, right, bottom, or left of the content area independently. The padding clears an area around the content and is transparent.
  3. Border: added around the top, right, bottom, or left of the padding independently.
  4. Margin: added to the top, right, bottom, or left of the border. The margin is transparent.

When should you use padding and margin?

Padding is used to control the size of a box without adjusting the size of the content inside the box.

Margin is used to control the space between boxes.

Calculate the box size

box width =
content area width +
(padding left + padding right) +
(border left + border right) +
(margin left + margin right)

box height =
content area height +
(padding top + padding bottom) +
(border top + border bottom) +
(margin top + margin bottom)

Apply box model properties

CSS
h1 {
  padding-top: 6px;
  padding-right: 3px;
  padding-bottom: 1px;
  padding-left: 0;
}

h1 {
  padding: 6px 3px 1px 0;
}

h1 {
  border-width: 6px;
  border-style: solid;
  border-color: black;
}

h1 {
  border: 6px solid black;
}

h1 {
  margin-top: 6px;
  margin-right: 2px;
  margin-bottom: 3px;
  margin-left: 0;
}

h1 {
  margin: 6px 2px 3px 0;
}

Browser defaults

Description

Browsers actually have a default styleheet for when no custom styles are set. To reset default styles (padding, border, margin) in order to set the box model properties by yourself include each tag you want to reset in CSS as shown below.

CSS
/* Reset browser defaults */
html, body, div, span, applet, object, iframe, h1, h2, h3, h4,
h5, h6, p, blockquote, pre, a, abbr, acronym, address, big,
cite, code, del, dfn, img, ins, kbd, q, s, samp, small,
strike, sub, sup, tt, var, u, dl, dt, dd, ol, ul, li,
fieldset, form, label, legend, table, caption, tbody, tfoot,
thead, tr, th, td, article, aside, canvas, details, embed,
figure, figcaption, footer, header, hgroup, menu, nav, output,
ruby, section, summary, time, mark, audio, video, em, i, b,
strong {
  margin: 0;
  padding: 0;
  border: 0;
  font-size: 100%;
  font: inherit;
  vertical-align: baseline;
}

It's important to start from a baseline and not let the browser determine what your defaults are. You need to be sure that the CSS you're writting is affecting the layout.

Display Types

Description

The display property specifies the type of box used for an element.

Syntax

CSS
element {
  display: none | inline | block | inline-block
}

Block Elements

Their content takes up the entire width of the container that they are in. These type of elements can have a width and height set on them and they respect top, right, bottom, and left margin/padding. Examples of block-level tags are <div>, <p>, <ul>, <ol>, <li>, <h1>, ..., <h6>.

Inline Elements

Their content does not take up more width that it needs. These elements are typically found within block-level elements. Inline elements cannot have a width and height set on them. Also these type of elements allow other elements to sit to their left and right. Inline elements respect only left & right margin/padding. Examples of inline-level elements are <span>, <a>, <img>, <input>, <label>, <strong>, <em>.

Inline-block Elements

Their content does not take up more width that it needs and they allow other elements to sit to their left and right (same flow as inline elements). However, unlike inline elements they behave the same as block elements. For example they can have a width and height set on them and they respect not only left & right margin/padding but also top & bottom margin/padding.

Centering Content

Centering an entire block element

If you want to center an entire block-level heading content element(e.g. h1-h6) you just need to set text-align to center.

HTML
<div>
  <h2>HEADING</h2>
</div>
CSS
div {
  border: 5px solid red;
}

div h2 {
  color: blue;
  text-align: center;
}
Result

HEADING

If you want to center an entire block-level sectioning content element(e.g. <div>, <section>), then you need to set the left and right margins of that element to auto. Be aware that center aligning has no effect if the width property is not set (or set to 100%).

HTML
<div>
  &nbsp;
</div>
CSS
div {
  width: 50%;
  margin: 0 auto;
}
Result
 

Centering inline and inline-block elements

You can center inline and inline-block elements within a block-level parent element by setting text-align to center.

HTML
<div>
  <img src="images/avatar.png" alt="avatar" width=50>
</div>
CSS
div {
  border: 5px solid red;
  text-align: center;
}
Result

Box Sizing

Description

The box-sizing property changes the default CSS box model, which is used to calculate widths and heights of given elements.

Content-box(default)

HTML
<div class="content-box">
</div>
CSS
.content-box {
  border: 10px solid black;
  margin: 20px;
  padding: 30px;
  width: 200px;
  height: 100px; }
Result

CONTENT

rendered box width = 200px(content) + 60px(padding) + 20px(border) = 280px

rendered box height = 100px(content) + 60px(padding) + 20px(border) = 180px

Padding-box*

<div class="padding-box">
</div> .padding-box {
  border: 10px solid black;
  margin: 20px;
  padding: 30px;
  box-sizing: padding-box;
  width: 200px;
  height: 100px;
}
Result

CONTENT

rendered box width = 200px[140px(content)+60px(padding)] + 20px(border) = 220px

rendered box height = 100px[40px(content)+60px(padding)] + 20px(border) = 120px

*This API is depricated and it should no longer be used.

Border-box

<div class="border-box">
</div> .border-box {
  border: 10px solid black;
  margin: 20px;
  padding: 30px;
  box-sizing: border-box;
  width: 200px;
  height: 100px;
}
Result

CONTENT

rendered box width = 200px[120px(content)+60px(padding)+20px(border)] = 200px

rendered box height = 100px[20px(content)+60px(padding)+20px(border)] = 100px

Overflow property

Syntax

CSS
overflow: visible / auto / hidden / scroll

where:

  • visible (default): Allows content to extend beyond container boundaries.
  • auto: The container stays the size that we set and adds a scrollbar as needed when content overflows.
  • hidden: Hides content that extends beyond container boundaries.
  • scroll: Adds a scrollbar at all times, even if it isn't needed.

Position property

Syntax

CSS
position: static / relative / absolute / fixed

where:

  • static (default): Normal behavior, the element is laid out in its current position in the normal flow.
  • relative: The element is laid out in its current position in the normal flow and then shifted via top/right/bottom/left.
  • absolute: The element is taken out of the normal flow for manual positioning relative to its closest positioned ancestor if any, or otherwise relative to the initial containing block.
  • fixed: The element is positioned to a specific place in the window, and does not move when scrolled.

Note: A positioned element is an element whose is either in relative, absolute, or fixed position.

Examples

  1. relative

    HTML
    <div>
      <p>
        Pythagorean equation: a<sup>2</sup> + b<sup>2</sup> = c<sup>2</sup>
      </p>
    </div>

    Note: The <sup> tag defines superscript text.

    CSS
    div sup {
      font-size: 75%;
      position: relative;
      top: -5px;
    }
    Result

Z-index

Description

By default an element that appears later in HTML, is rendered above an earlier element (cascade order). However if the earlier element has a larger z-index value, it will be the one on top. The z-index only works on positioned elements (absolute, relative, fixed).

Syntax

CSS
z-index: integer

Example

HTML
<div id="z-index">
  <img id="img-1" src="images/number-1.png" alt="number-1" width="300">
  <img id="img-2" src="images/number-2.png" alt="number-2" width="300">
<div>
  • CSS
    #z-index {
      position: relative;
      border: 5px solid red;
      height: 500px;
    }

    #img-1 {
      position: absolute;
      left: 200px;
      top: 20px;
      z-index: 0;
    }

    #img-2 {
      position: absolute;
      left: 400px;
      top: 50px;
      z-index: 0;
    }
    Result
