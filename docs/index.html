About

Welcome to HTML5Tech, an ever-growing web site with fundamental resources about HTML5 and friends.

I've started this site while learning web development. The site is fully responsive and does not use any external libraries or frameworks. It has been developed from scratch using plain HTML5, CSS3 and JavaScript. The content is presented in a readable, easier to understand fashion to help beginners in the field of front-end web development. Each article covers a major topic in a client-side technology therefore this site can be considered an informational blog.

Aim

I've decided to start this page, to help myself and other newbies in the field of web development. From this site you can learn the fundamental concepts of client-side technologies (i.e. HTML, CSS, Javascript) or you can simply use the site as a reference.

Get involved

If you're interested in contributing to the website development, head over to the site's repository on github. Any questions or suggestions are more than welcomed.
