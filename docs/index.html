Welcome

Welcome to HTML5Tech, an ever-growing website with fundamental resources about HTML5 and friends.

The aim of this website is to help beginners in the field of web development to learn the basics through helpful resources, detailed introductions, and visual examples.

HTML5Tech is primarily a reference site on web fundamentals and the core technologies of the web but also includes documentation for popular libraries and web frameworks.

The long-term goal of this site is to be turned into a progressive web application that will load fast regardless of network state and be functional even offline, providing a comprehensive reference guide to web developers.

The purpose of this site is to share knowledge. You are free to copy and redistribute the material in any medium or format and adapt it to your needs. Note that full correctness of all content is not guaranteed but any comments or suggestions would be greatly appreciated.

HTML

The markup language of the web.

The HTML we use today was developed from the prototype written by Tim Berners-Lee (inventor of the World Wide Web) in 1992.

Learn more

CSS

The stylesheet language of the web.

Håkon Wium Lie proposed the concept of Cascading Style Sheets in 1994. In 1996, Wium Lie teamed up with Bert Bos, the creator of Argo web browser. Together they used the browser as a testbed for the style sheet proposal.

Learn more

JavaScript

The programming language of the web.

Brendan Eich created the JavaScript programming language in 1995. He co-founded the Mozilla project, the Mozilla Foundation, and the Mozilla Corporation.

jQuery

A JavaScript library that simplifies the development of web pages.

John Resig created jQuery in 2006 and currently serves as the lead developer of the library.
