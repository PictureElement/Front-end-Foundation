Border Radius

Description

The border-radius property is used to add rounded corners to an element's border.

Examples

  1. top-left&bottom-right&top-right&bottom-left

    HTML
    <div class="border-radius-1">
    </div>
    CSS
    .border-radius-1 {
      height: 100px;
      width: 200px;
      border: 5px solid red;
      border-radius: 25px;
    }
    Result

  2. top-left&bottom-right | top-right&bottom-left

    HTML
    <div class="border-radius-2">
    </div>
    CSS
    .border-radius-2 {
      height: 100px;
      width: 200px;
      border: 5px solid red;
      border-radius: 25px 50px;
    }
    Result

  3. top-left | top-right&bottom-left | bottom-right

    HTML
    <div class="border-radius-3">
    </div>
    CSS
    .border-radius-3 {
      height: 100px;
      width: 200px;
      border: 5px solid red;
      border-radius: 20px 40px 60px;
    }
    Result

  4. top-left | top-right | bottom-right | bottom-left

    HTML
    <div class="border-radius-4">
    </div>
    CSS
    .border-radius-4 {
      height: 100px;
      width: 200px;
      border: 5px solid red;
      border-radius: 10px 20px 30px 40px;
    }

    /* equivalent to: */

    .border-radius-4 {
      border-top-left-radius: 10px;
      border-top-right-radius: 20px;
      border-bottom-right-radius: 30px;
      border-bottom-left-radius: 40px;
    }
    Result

  5. top-left&bottom-right&top-right&bottom-left(horizontal radii)/ top-left&bottom-right&top-right&bottom-left(vertical radii)

    HTML
    <div class="border-radius-5">
    </div>
    CSS
    .border-radius-5 {
      height: 100px;
      width: 200px;
      border: 5px solid red;
      border-radius: 30px/60px;
    }

    equivalent to:

    .border-radius-5 {
      border-top-left-radius: 30px 60px;
      border-top-right-radius: 30px 60px;
      border-bottom-right-radius: 30px 60px;
      border-bottom-left-radius: 30px 60px;
    }
    Result

  6. top-left&bottom-right&top-right&bottom-left(horizontal radii)/ top-left&bottom-right&top-right&bottom-left(vertical radii)

    HTML
    <div class="border-radius-6">
    </div>
    CSS
    .border-radius-6 {
      height: 100px;
      width: 200px;
      border: 5px solid red;
      border-radius: 60px/30px;
    }

    equivalent to:

    .border-radius-6 {
      border-top-left-radius: 60px 30px;
      border-top-right-radius: 60px 30px;
      border-bottom-right-radius: 60px 30px;
      border-bottom-left-radius: 60px 30px;
    }
    Result

  7. Oval shape

    HTML
    <div class="border-radius-7">
    </div>
    CSS
    .border-radius-7 {
      height: 100px;
      width: 200px;
      border: 5px solid red;
      border-radius: 50%;
    }
    Result

  8. Circle shape

    HTML
    <div class="border-radius-8">
    </div>
    CSS
    .border-radius-8 {
      height: 200px;
      width: 200px;
      border: 5px solid red;
      border-radius: 50%;
    }
    Result

Box Shadow

Description

The box-shadow property enables you to cast one or more shadows from the frame of almost any element.

Syntax

box-shadow: inset offset-x offset-y blur-radius spread-radius color

where:

  • inset (optional): if not specified (default), a drop shadow is created rather than an inset shadow.
  • offset-x: moves the shadow along the x-axis (left & right).
  • offset-y: moves the shadow along the y-axis (up & down).
  • blur-radius (optional): changes the blur amount of the shadow. The larger the value, the bigger the blur, so the shadow becomes bigger and lighter. Negative values are not allowed. If not specified the value will be 0 (shadow's edge is sharp).
  • spread-radius (optional): causes the shadow to expand (grow bigger) or shrink. In other words, it changes the size of the shadow.
  • color (optional): specifies the color of the shadow.

Examples

  1. Sharp shadow edge (no blur):

    box-shadow: 5px 10px black;

  2. Sharp shadow edge (no blur):

    box-shadow: -5px 10px black;

  3. Sharp shadow edge (no blur):

    box-shadow: 5px -10px black;

  4. Sharp shadow edge (no blur):

    box-shadow: -5px -10px black;

  5. Specify a blur radius:

    box-shadow: 5px 10px 5px black;

  6. Specify a spread radius (expand shadow):

    box-shadow: 5px 10px 5px 5px black;

  7. Specify a spread radius (shrink shadow):

    box-shadow: 5px 10px 5px -5px black;

  8. Specify a spread radius instead of blur-radius:

    box-shadow: 5px 10px 0 5px black;

  9. Inset and outset/drop shadow combination:

    box-shadow:
      inset 5px 10px 5px gray,
      5px 10px 5px black;

  10. Specify multiple box shadows:

    box-shadow:
      0 0 10px black,
      /* x: right, y: down */
      10px 5px yellow,
      /* x: left, y: down */
      -10px 5px red,
      /* x: left, y: up */
      -10px -5px blue,
      /* x: right, y: up */
      10px -5px lime;
