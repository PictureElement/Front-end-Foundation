Border Radius

Description

The border-radius property is used to add rounded corners to an element's border.

Examples

  1. top-left&bottom-right&top-right&bottom-left

    HTML
    <div class="border-radius-1">
    </div>
    CSS
    .border-radius-1 {
      height: 100px;
      width: 200px;
      border: 5px solid red;
      border-radius: 25px;
    }
    Result

  2. top-left&bottom-right | top-right&bottom-left

    HTML
    <div class="border-radius-2">
    </div>
    CSS
    .border-radius-2 {
      height: 100px;
      width: 200px;
      border: 5px solid red;
      border-radius: 25px 50px;
    }
    Result

  3. top-left | top-right&bottom-left | bottom-right

    HTML
    <div class="border-radius-3">
    </div>
    CSS
    .border-radius-3 {
      height: 100px;
      width: 200px;
      border: 5px solid red;
      border-radius: 20px 40px 60px;
    }
    Result

  4. top-left | top-right | bottom-right | bottom-left

    HTML
    <div class="border-radius-4">
    </div>
    CSS
    .border-radius-4 {
      height: 100px;
      width: 200px;
      border: 5px solid red;
      border-radius: 10px 20px 30px 40px;
    }

    /* equivalent to: */

    .border-radius-4 {
      border-top-left-radius: 10px;

      border-bottom-left-radius: 40px;
    }

    HTML

    equivalent to:

Box Shadow

Description

The box-shadow property enables you to cast one or more shadows from the frame of almost any element.

Syntax

box-shadow: inset offset-x offset-y blur-radius spread-radius color

where:

  • inset (optional): if not specified (default), a drop shadow is created rather than an inset shadow.
  • offset-x: moves the shadow along the x-axis (left & right).
  • offset-y: moves the shadow along the y-axis (up & down).
  • blur-radius (optional): changes the blur amount of the shadow. The larger the value, the bigger the blur, so the shadow becomes bigger and lighter. Negative values are not allowed. If not specified the value will be 0 (shadow's edge is sharp).
  • spread-radius (optional): causes the shadow to expand (grow bigger) or shrink. In other words, it changes the size of the shadow.
  • color (optional): specifies the color of the shadow.

Examples

  1. Sharp shadow edge (no blur):

    box-shadow: 5px 10px black;

  2. Sharp shadow edge (no blur):

    box-shadow: -5px 10px black;

  3. Sharp shadow edge (no blur):

    box-shadow: 5px -10px black;

  4. Sharp shadow edge (no blur):

    box-shadow: -5px -10px black;

  5. Specify a blur radius:

    box-shadow: 5px 10px 5px black;

  6. Specify a spread radius (expand shadow):

    box-shadow: 5px 10px 5px 5px black;

  7. Specify a spread radius (shrink shadow):

    box-shadow: 5px 10px 5px -5px black;

  8. Specify a spread radius instead of blur-radius:

    box-shadow: 5px 10px 0 5px black;

  9. Inset and outset/drop shadow combination:

    box-shadow:
      inset 5px 10px 5px gray,
      5px 10px 5px black;

  10. Specify multiple box shadows:

    box-shadow:
      0 0 10px black,
      /* x: right, y: down */
      10px 5px yellow,
      /* x: left, y: down */
      -10px 5px red,
      /* x: left, y: up */
      -10px -5px blue,
      /* x: right, y: up */
      10px -5px lime;

Text Shadow

Description

The text-shadow property adds shadows to text.

Syntax

text-shadow: offset-x offset-y blur-radius color
/* OR */
text-shadow: color offset-x offset-y blur-radius

where:

  • offset-x: position of horizontal shadow.
  • offset-y: position of vertical shadow.
  • blur-radius (optional): controls the blur amount. The higher this value, the bigger the blur and the shadow becomes wider and lighter.
  • color (optional): specifies the color of the shadow. This value can be specified either before or after the offset values.

Note: You can apply multiple text shadows, using a comma-separated list.

Examples

  1. Sharp shadow (no blur):

      color: black;
      text-shadow: 5px 5px #a0a0a0;

Box Sizing

Description

The box-sizing property changes the default CSS box model, which is used to calculate widths and heights of given elements.

Content-box(default)

HTML
<div class="content-box">
</div>
CSS
.content-box {
  border: 10px solid black;
  margin: 20px;
  padding: 30px;
  width: 200px;
  height: 100px; }
Result

CONTENT

rendered box width = 200px(content) + 60px(padding) + 20px(border) = 280px

rendered box height = 100px(content) + 60px(padding) + 20px(border) = 180px

Padding-box*

<div class="padding-box">
</div> .padding-box {
  border: 10px solid black;
  margin: 20px;
  padding: 30px;
  box-sizing: padding-box;
  width: 200px;
  height: 100px;
}
Result

CONTENT

rendered box width = 200px[140px(content)+60px(padding)] + 20px(border) = 220px

rendered box height = 100px[40px(content)+60px(padding)] + 20px(border) = 120px

*This API has not been standardized

Border-box

<div class="border-box">
</div> .border-box {
  border: 10px solid black;
  margin: 20px;
  padding: 30px;
  box-sizing: border-box;
  width: 200px;
  height: 100px;
}
Result

CONTENT

rendered box width = 200px[120px(content)+60px(padding)+20px(border)] = 200px

rendered box height = 100px[20px(content)+60px(padding)+20px(border)] = 100px

Multiple Backgrounds

Description

CSS3 allows you to apply multiple backgrounds to an element. Backgrounds are stacked on top of each other. The first background you provide is on top of the stack and the last background listed in the back.

Syntax

  1. Specify individual properties:
    CSS
    .myclass {
      background-image: value1, ..., valueN;
      background-position: value1, ..., valueN;
      background-size: value1, ..., valueN;
      background-repeat: value1, ..., valueN;
    }
  2. Shorthand:
    CSS
    .myclass {
       background:
        bg-image1 bg-position1 bg-size1 bg-repeat1 bg-color1,
        ...
        bg-imageN bg-positionN/bg-sizeN bg-repeatN bg-colorN;
    }

    Note: If one of the properties in the shorthand syntax is the bg-size property, you must use a slash(/) to separate it from the bg-position property (e.g. 10px 20px/50px 40px, top/cover, etc.)

Example

HTML
<div class="mult-bg-1">
</div>
CSS
.mult-bg-1 {
  border: 5px solid black;
  height: 300px;
  background-image:
    url(images/avatar.png),
    url(images/lightning.svg),
    linear-gradient(to left, #dc2424, #4a569d);
  background-position: 50% 20%, center, center;
  background-size: 150px, contain, contain;
  background-repeat: no-repeat, no-repeat, no-repeat;
}
/* OR (Shorthand) */
.mult-bg-1 {
  border: 5px solid black;
  height: 300px;
  background:
    url(images/avatar.png) 50% 20%/150px no-repeat,
    url(images/lightning.svg) center/contain no-repeat,
    linear-gradient(to left, #dc2424, #4a569d) center/contain no-repeat;
}
Result
