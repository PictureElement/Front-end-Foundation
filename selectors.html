Selectors

PictureElement
Mar 14, 2017

Descendant Selector

Description

To apply a CSS rule to a single tag or a subset of tags we use a descendant selector.

Example

CSS

ul li {
font-size: 24px;
}

The CSS rule above selects only <li> tags that are children of <ul> tags.

Pseudo-selector

Description

A modifier that can be added to a selector to select a tag only when a certain condition has occured.

Example #1

CSS

a {
text-decoration: none;
}

a:hover {
text-decoration: underline;
color: darkred;
}

The "hover" pseudo-selector is used. The rule is only going to be applied when the mouse is hovering over the link.

Example #2

HTML

...
<ul>
<li>2 egss</li>
<li>1 pound of sugar</li>
</ul>
...

CSS

ul li:first-child {
color: red;
}

The "first-child" pseudo-selector is applied to narrow down the amount of child tags selected. The style is applied only on the "2 eggs" line.
