Selectors

PictureElement
Mar 14, 2017

Descendant Selector

Description

To apply a CSS rule to a single tag or a subset of tags we use a descendant selector.

Example

CSS
ul li {
  font-size: 24px;
}

The CSS rule above selects only <li> tags that are children of <ul> tags.

Pseudo-selector

Description

A modifier that can be added to a selector to select a tag only when a certain condition has occured.

Examples

  1. A "hover" pseudo-selector is used. The rule is only going to be applied when the mouse is hovering over the link:

    CSS
    a {
      text-decoration: none;
    }

    a:hover {
      text-decoration: underline;
      color: darkred;
    }

  2. The "first-child" pseudo-selector is applied to narrow down the amount of child tags selected. The style is applied only on the "2 eggs" line:

    HTML
    ...
    <ul>
      <li>2 egss</li>
      <li>1 pound of sugar</li>
    </ul>
    ...
    CSS
    ul li:first-child {
      color: red;
    }

Class selector

Description

The .class selector selects elements with a specific class attribute.

Syntax

CSS
.class {
  declarations;
}

Examples

  1. Select and style only one <ul>:

    HTML
    <ul class="nav">
      <li><a href="home.html">Home</a></li>
      <li><a href="recipes.html">Recipes</a></li>
      <li><a href="suggest.html">Suggest</a></li>
    </ul>
    CSS
    .nav {
      declarations;
    }

  2. Change the style of only <a> tags that are inside of a <ul> tag with a class of nav:

    HTML
    <ul class="nav">
      <li><a href="home.html">Home</a></li>
      <li><a href="recipes.html">Recipes</a></li>
      <li><a href="suggest.html">Suggest</a></li>
    </ul>
    CSS
    .nav a {
      declarations;
    }

  3. We want only the navigation list to display inline:

    HTML
    <ul class="nav">
      <li><a href="home.html">Home</a></li>
      <li><a href="recipes.html">Recipes</a></li>
      <li><a href="suggest.html">Suggest</a></li>
    </ul>

    <ul>
      <li>2 eggs</li>
      <li>1 pound of brown sugar</li>
      <li>3 sticks of butter</li>
    </ul>
    CSS
    /*
    Select <li> elements that are children of <ul> elements with a class of nav and make them display inline.
    */
    .nav li {
      display: inline;
    }

  4. We need a way to access the creator paragraph tag in our page in order to add some specific properties. To achieve that we have to add a class called creator to each paragraph tag that contains the creator name:

    HTML
    <p class="creator">creator: Marios Sofokleous</p>
    CSS
    .creator {
      declarations;
    }
