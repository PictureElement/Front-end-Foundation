Staying DRY

Description

DRY stands for Don't Repeat Yourself

Examples

  1. Elements that fall within the same parent container

    • A clumsy solution:

      HTML

      <section>
        <h1>Heading</h1>
        <p>Paragraph</p>
        <a>Link</a>
      </section>

      CSS

      h1 {
        font-family: Arial, Helvetica;
      }

      p {
        font-family: Arial, Helvetica;
      }

      a {
        font-family: Arial, Helvetica;
      }

    • A clean solution - Every element inherits the style from the container:

      HTML

      <section>
        <div class="container">
          <h1>Heading</h1>
          <p>Paragraph</p>
          <a>Link</a>
        </div>
      </section>

      CSS

      .container {
        font-family: Arial, Helvetica;
      }

  2. Elements that do not fall within the same parent container

    HTML

    <header>
      <h1>Header</h1>
    <section>
      <p>Paragraph</p>
    </section>
    <aside>
      <a>Link</a>
    </aside>

    • A clumsy solution:

      CSS

      h1 {
        font-size: 15px;
      }

      p {
        font-size: 15px;
      }

      a {
        font-size: 15px;
      }

    • A clean solution - Use of multiple selectors:

      CSS

      h1, p, a {
        font-size: 15px;
      }

  3. Selector abstraction 1

    • A clumsy solution:

      HTML

      <header>
        ...
      </header>
      <footer>
        ...
      </footer>

      CSS

      header {
        background: black;
        ...
        color: white;
      }

      footer {
        background: black;
        ...
        color: white;
      }

    • A clean solution:

      HTML

      <header>
        <div class="dark-bg">
          ...
        </div>
      </header>

      <footer>
        <div class="dark-bg">
          ...
        </div>
      </footer>

      CSS

      .dark-bg {
        background: black;
        ...
        color: white;
      }

    What if we want the footer to inherit the same properties (.dark-bg class) as the header except the background color?

    • A clumsy solution:

      HTML

      <header>
        <div class="dark-bg">
          ...
        </div>
      </header>

      <footer>
        <div class="white-bg">
          ...
        </div>
      </footer>

      CSS

      .dark-bg {
        background: black;
        ...
        color: white;
      }
      .white-bg {
        background: white;
        ...
        color: white;
      }

    • A clean solution: The footer element will be pulling its styles from both the dark-bg class and the white-bg subclass. It's important to note that order matters here (cascade order) in order for the white-bg class to be able to overwrite the background color of the dark-bg class.

      HTML

      <header>
        <div class="dark-bg">
          ...
        </div>
      </header>

      <footer>
        <div class="dark-bg white-bg">
          ...
        </div>
      </footer>

      CSS

      .dark-bg {
        background: black;
        ...
        color: white;
      }
      .white-bg {
        background: white;
      }

  4. Use of shorthand properties

    • A clumsy solution:

      CSS

      .my-class {
        margin-top: 10px;
        margin-right: 20px;
        margin-bottom: 10px;
        margin-left: 20px;
      }

    • A clean solution:

      CSS

      .my-class {
        margin: 10px 20px 10px 20px;

    Property shorthands:

    • font: size/line-height weight style font-family;
      font: 15px/20px bold italic arial;
    • background: color image repeat x-pos y-pos;
      background: black url(img.png) no-repeat center top;
    • list-style: bullet-style position image;
      list-style: disc inside none;
    • margin/padding: top right bottom left;
      margin/padding: 10px 20px 0 15px;
      margin/padding: top&bottom right&left;
      margin/padding: 10px 20px;
      margin/padding: top right&left bottom;
      margin/padding: 10px 20px 10px;
    • border: width line-style color;
      border: 1px solid black;
