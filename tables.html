A typical table

      <td colspan="2">1.21053</td>
    </tr>
  </tfoot>
</table>

where:

  • <caption>: A table caption element that provides information about the table.
  • <span>: A generic inline element that does not represent anything and is often used to style a part of a text. It should be used only when no other semantic element is appropriate (just like <div>).
  • <thead>: Groups the header content in a table.
  • <tfoot>: Groups the footer content in a table.
  • <tbody>: Groups the body content in a table.
  • <tr>: Defines a table row.
  • <th>: Defines a header cell that contains header information.
  • <td>: Defines a standard cell that contains data.
  • scope: This attribute specifies whether a header cell refers to a column, row, or group of columns or rows.
  • rowspan: This attribute specifies the number of rows a cell should extend.
  • colspan: This attribute specifies the number of columns a cell should extend.

Styling our table

CSS

/* Spacing & layout styling */
.table-1 {
  table-layout: fixed;
  width: 100%;
  border-collapse: collapse;
  border: 2px solid black;
}

.table-1 thead th:nth-child(1) {
  width: 25%;
}

.table-1 thead th:nth-child(2) {
  width: 25%;
}

.table-1 thead th:nth-child(3) {
  width: 25%;
}

.table-1 thead th:nth-child(4) {
  width: 25%;
}

.table-1 th, .table-1 td {
  padding: 10px;
}

/* Font styling & text alignment */
.table-1 {
  color: black;
  font-size: 15px;
  font-weight: 100;
  font-family: Lato, Helvetica, Arial;
  margin-top: 20px;
}

.table-1 thead {
  color: white;
  font-size: 20px;
  font-weight: bold;
  font-family: Arvo, Times, "Times New Roman";
  letter-spacing: 2px;
}

.table-1 tfoot th {
  color: black;
  font-size: 20px;
  font-weight: bold;
  font-family: Arvo, Times, "Times New Roman";
  letter-spacing: 2px;
}

.table-1 td {
  letter-spacing: 1px;
}

.table-1 tbody td {
  text-align: center;
  vertical-align: middle;
}

.table-1 tfoot th {
  text-align: center;
}

.table-1 tfoot td {
  text-align: center;
}

/* Borders */
.table-1 thead th {
  border: 2px solid black;
}

.table-1 tbody td {
  border: 2px solid black;
}

.table-1 tfoot th {
  border: 2px solid black;
}

.table-1 tfoot td {
  border: 2px solid black;
}

/* Backgrounds & text shadows */
.table-1 thead {
  background: linear-gradient(to bottom, royalblue, steelblue);
  text-shadow: 1px 1px 1px black;
}

.table-1 tbody tr:nth-child(odd) {
  background-color: gainsboro;
}

.table-1 tbody tr:nth-child(even) {
  background-color: gray;
}

.table-1 tfoot tr:nth-child(even) {
  background-color: red;
}

.table-1 tfoot tr:nth-child(odd) {
  background-color: green;
}

/* Caption styling */
.table-1 caption {
  font-family: Arvo, Times, "Times New Roman";
  font-weight: bold;
  font-size: 12px;
  caption-side: bottom;
  letter-spacing: 1px;
  text-align: left;
  margin: 10px;
}

.table-1 caption span {
  font-weight: normal;
  font-style: italic;
}

Notes:

  • table-layout: fixed makes the table behave more predictable by preventing table columns to be sized according to their content. With table-layout: fixed we can size the columns manually.
  • border-collapse: collapse eliminate spacing between table borders and make the borders collapse down into one.
  • letter-spacing specifies spacing behavior between text characters. Letter spacing aids readability.
  • vertical-align specifies the vertical alignment of text inside table cells.
  • text-shadow specifies a text shadow which makes text to contrast well with backgrounds(i.e. images, gradients).
  • :nth-child(n) is a pseudo-class that matches every element that is the nth child of its parent. n can be a number, a keyword(e.g. odd, even), or a formula(e.g. 2n, 2n-1)
