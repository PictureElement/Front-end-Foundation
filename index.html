Updated Tags

"i" tag

  • Description: represents text in an "alternate voice" or "mood".

  • Example:

    I hope this works, he thought.

"b" tag

  • Description: represents "stylistically offset" text.

  • Example:

    The event takes place this upcoming Saturday, and over 3,000 people have already registered.

"em" tag / "strong" tag

  • Description ("em" tag): stress emphasis.

    Description ("strong" tag): strong importance.

  • Example:

    Make sure to sign up before the day of the event, September 16, 2013.

Updated Elements

Example:

<!DOCTYPE html> (1)

<html>

  <head>

    <meta charset="UTF-8"> (2)

    <link rel="stylesheet" href="foo.css"> (3)

    <script src="foo.js"></script> (4.1)

    <script>JS code</script> (4.2)

    ...

  </head>

  ...

</html>

Description:

  • DOCTYPE (1): set the HTML version that the browser should use. If you not specify a version number, you're telling the browser to use the latest version.

  • Meta declaration (2): specify character encoding.

  • Link tag (3): reference external CSS files.

  • Script tag (4.1): reference external JS files.

  • Script tag (4.2): run integrated JS code.
