The Box Model

PictureElement
Mar 15, 2017

Description

Every tag shown in the body is contained in an invisible rectangle called the box. The box model is a way to describe the borders and spacing in between the boxes of each tag. The box model allows us to add a border around elements, and to define space between elements.

Properties

  1. Content area: actual content of the box, where text and images appear.
  2. Padding: added to the top, right, bottom, or left of the content area independently. The padding clears an area around the content and is transparent.
  3. Border: added around the top, right, bottom, or left of the padding independently.
  4. Margin: added to the top, right, bottom, or left of the border. The margin is transparent.

When should you use padding and margin?

Padding is used to control the size of a box without adjusting the size of the content inside the box.

Margin is used to control the space between boxes.

Calculate the box size

box width =
content area width +
(padding left + padding right) +
(border left + border right) +
(margin left + margin right)

box height =
content area height +
(padding top + padding bottom) +
(border top + border bottom) +
(margin top + margin bottom)

Apply box model properties

CSS

h1 {
  padding-top: 6px;
  padding-right: 3px;
  padding-bottom: 1px;
  padding-left: 0;
}

h1 {
  padding: 6px 3px 1px 0;
}

h1 {
  border-width: 6px;
  border-style: solid;
  border-color: black;
}

h1 {
  border: 6px solid black;
}

h1 {
  margin-top: 6px;
  margin-right: 2px;
  margin-bottom: 3px;
  margin-left: 0;
}

h1 {
  margin: 6px 2px 3px 0;
}

Browser defaults

Description

Browsers actually have a default styleheet for when no custom styles are set. To reset default styles (padding, border, margin) in order to set the box model properties by yourself include each tag you want to reset in CSS as shown below.

CSS

/* Reset browser defaults */
html, body, div, span, applet, object, iframe, h1, h2, h3, h4,
h5, h6, p, blockquote, pre, a, abbr, acronym, address, big,
cite, code, del, dfn, img, ins, kbd, q, s, samp, small,
strike, sub, sup, tt, var, u, dl, dt, dd, ol, ul, li,
fieldset, form, label, legend, table, caption, tbody, tfoot,
thead, tr, th, td, article, aside, canvas, details, embed,
figure, figcaption, footer, header, hgroup, menu, nav, output,
ruby, section, summary, time, mark, audio, video, em, i, b,
strong {
  margin: 0;
  padding: 0;
  border: 0;
  font-size: 100%;
  font: inherit;
  vertical-align: baseline;
}

It's important to start from a baseline and not let the browser determine what your defaults are. You need to be sure that the CSS you're writting is affecting the layout.

Block-level & Inline-level Tags

Block-level Tags

Their content takes up the entire width of the container that they are in. Examples of block-level tags are <p>, <ul>, <ol>, <li>, <h1>, ..., <h6>.

Inline-level Tags

Their content does not take up more width that it needs. Examples of inline-level tags are <span>, <a>, <img>, <input>, <label>.

Turning block-level into inline-level and vice versa

The display property specifies the type of box used for an HTML element.

CSS

/* Display an element as an inline element */
element {
  display: inline;
}

/* Display an element as a block element */
element {
  display: block;
}

Centering Content

Center an entire block-level tag

If you want to center an entire block-level tag and that tag is fixed width, then you can set the left and right margins of that tag to auto.

CSS

.class {
  margin: 0 auto 0 auto;
}

element {
  margin: 0 auto 0 auto;
}

h1 {
  margin: 0 auto 0 auto;
}

Center the content that's inside of a block-level tag

If you want to center the content that's inside of a block-level tag, then you want to set the text-align property to center.

CSS

h1 {
  text-align: center;
}
