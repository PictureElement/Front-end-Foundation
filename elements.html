Division

Description

The <div> element is a generic block-level container used to group related content. This element does not inherently represent anything, in other words it hasn't got any semantic meaning. The <div> element should be used only when no other semantic element(e.g. <article>, <section>) is appropriate. This element is mainly used for styling purposes.

Example

Style the byline(author, date, avatar) of an article:

HTML

<div class="byline-wrapper">
  <img src="images/avatar.png" alt="avatar" width="40px">
  <p>
    <a href="mailto:pictureelement@yandex.com">PictureElement</a>
    <br>
    <time datetime="2017-02-28">Feb 28, 2017</time>
  </p>
</div>

CSS

.byline-wrapper {
  declarations;
}

.byline-wrapper img {
  declarations;
}

.byline-wrapper p {
  declarations;
}
