RGB Color Model

Description

The RGB color model adds red, green, and blue together in various ways to reproduce a broad array of colors. In this model, a color is described by indicating how much of each of the red, green, and blue is included. The color is expressed as an RGB triplet (r,g,b). In computers, the red, green, and blue component values are stored as integer numbers in the range 0-255. These are often represented as either decimal or hexadecimal numbers.

rgb()

Hexadecimal Notation

#RRGGBB: six-digit RGB hex notation
First two digits set the amount of red.
Next two digits set the amount of green.
Last two digits set the amount of blue.

#RGB: three-digit RGB hex notation
First digit sets the amount of red.
Second digit sets the amount of green.
Last digit sets the amount of blue.

#0ff and #00ffff represent the same color (Cyan).

Functional Notation

rgb(r,g,b)
rgb(pctr,pctg,pctb)

where:

  • 0 ≤ r,g,b ≤ 255, r,g,b ∈ ℕ
  • pctr,pctg,pctb: 0-100%

rgba()

Hexadecimal Notation

#RRGGBBaa: eight-digit RGB hex notation
First two digits set the amount of red.
Second two digits set the amount of green.
Third two digits set the amount of blue.
Last two digits set the transparency.

#RGBa: four-digit RGB hex notation
First digit sets the amount of red.
Second digit sets the amount of green.
Third digit sets the amount of blue.
Last digit sets the transparency.

#0ff0 and #00ffff00 represent the same color (0% opaque Cyan).
#0fff and #00ffffff represent the same color (full opaque Cyan).

Functional Notation

rgba(r,g,b,a)

where:

  • 0 ≤ r,g,b ≤ 255, r,g,b ∈ ℕ
  • 0 ≤ a ≤ 1, a ∈ ℜ, a means opacity: 0=transparent, 1=opaque

Examples

  1. #7403ab
    Red: 7x16 + 4x1 = 112(DEC)
    Green: 0x16 + 3x1 = 3(DEC)
    Blue: 10x16 + 11x1 = 171(DEC)
    *The color is some red, almost no green and more blue than red. The color is purplish.
  2. #fff = #ffffff
    Red: 255(DEC)
    Green: 255(DEC)
    Blue: 255(DEC)
    *The color is white.
  3. rgb(255,255,255)
    *The color is white.
  4. rgb(100%,100%,100%)
    *The color is white.
  5. rgba(255,255,255,1)
    *The color is full opaque white.
  6. rgba(255,255,255,0.2)
    *The color is 20% opaque white.

Usefull Info

  • Primary colors: Red, Green, Blue
  • Colors on the web: 256 possible combinations for each color channel => 256 * 256 * 256 = 16777216 available colors
  • Magenta(ff00ff): Red + Blue
  • Yellow(ffff00): Red + Green
  • Cyan(00ffff): Green + Blue
  • Black(000000): Absense of color
  • White(ffffff): All three primary colors at maximum intensities
  • Light gray: #D3D3D3
  • Gray: #808080
  • Dark gray: #A9A9A9

HSL Color Model

Description

Colors can also be defined via HUE, SATURATION, and LIGHTNESS, or HSL. Hue is represented as an angle of the color circle. By definition pure red=0(or 360), pure green=120, pure blue=240. Saturation and lightness are represented as percentages. 100% is full saturation, and 0% is a shade of gray. 100% ligthness is white, 0% lightness is black, and 50% lightness is normal.

hsl()

hsl(hue,saturation,lightness)

where:

  • 0 ≤ hue ≤ 360
  • saturation: 0-100%
  • lightness: 0-100%

hsla()

hsla(hue,saturation,lightness,a)

where:

  • 0 ≤ hue ≤ 360
  • saturation: 0-100%
  • lightness: 0-100%
  • 0 ≤ a ≤ 1, a ∈ ℜ, a means opacity: 0=transparent, 1=opaque

Color Wheel

Examples

  1. hsl(0, 100%, 50%)
    *The color is red
  2. hsl(60, 100%, 50%)
    *The color is yellow
  3. hsl(120, 100%, 50%)
    *The color is green.
  4. hsl(180, 100%, 50%)
    *The color is cyan.
  5. hsl(240, 100%, 50%)
    *The color is blue.
  6. hsl(300, 100%, 50%)
    *The color is magenta.
  7. hsla(300, 100%, 50%, 0.05)
    *The color is 5% opaque magenta.
  8. hsla(300, 100%, 50%, 0.5)
    *The color is 50% opaque magenta.
  9. hsla(300, 100%, 50%, 1)
    *The color is full opaque magenta.
