    Datalist

    Description

    The <datalist> element specifies a list of pre-defined options for an <input> element. Users will see a drop-down list of pre-defined options and auto-complete suggestions as they input data.

    Example:

    HTML5

    <form>
      Choose an OS:
      <input list="operating systems">
      <datalist id="operating systems">
        <option value="Windows">
        <option value="macOS">
        <option value="Linux">
        <option value="Solaris">
        <option value="FreeBSD">
      </datalist>
    </form>

    Notice: The "list" and "id" attributes need to be the same. The <input> element needs to know about the <datalist> options.
