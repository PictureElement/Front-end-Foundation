Common Mistakes

PictureElement
Mar 21, 2017

Using an element other than <li> inside a <ul> element.

HTML
<ul>
  <h2>...</h2>
  <li>...</li>
  <p>...</p>
  <li>...</li>
</ul>

You can however, put any element inside <li>, like so:

HTML
<ul>
  <li>
    h2>...</h2>
    p>...</p>
  </li>
  <li>...</li>
</ul>

Nesting a block-level element inside a <p> tag.

HTML
<p>
  <ul>
    ...
  </ul>
</p>
HTML
<p>
  <div>
    ...
  </div>
</p>

You can only nest inline elements such as <span>, <a>, <b>, <strong>, and <img> inside <p> tag. The <p> tag represents a paragraph. It cannot contain block-level elements including <p> itself.
