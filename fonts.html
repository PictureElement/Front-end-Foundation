Fonts

PictureElement
Mar 17, 2017

font-family property

Description

font-style changes can be applied with CSS. The font-family property is used to specify a list of fonts that should be used for an element. The property can hold several font names as a "fallback" system.

Example

CSS

element {
  font-family: Helvetica, Arial, "Times New Roman", sans-serif;
}

The first font in the list(i.e. Helvetica) represents the ideal font that the browser will try to load first. The Arial, Times New Roman, and sans-serif fonts are fallback fonts. If the browser can't find the ideal font, it will try to load the second font in the list and so on.

Notes:

  • Use quotes("") if fonts are multi-word.
  • Not all browsers and OS have all fonts installed by default.

After applying a main font to the element, the browser still uses default font styles. Just like we do with padding, borders, and margins, we can reset our font styles to a baseline default level in order for us to improve upon:

CSS

element {
  font-size: 100%;
  font: inherit;
}

font-style property

Description

The font-style property is mostly used to specify italic text.

Usage

CSS

element {
  font-style: value;
}

where value equals:

  • normal - text is shown normally
  • italic - text is shown in italics

font-weight property

Description

The font-weight property specifies the weight of a font.

Usage

CSS

element {
  font-weight: value;
}

where value equals:

  • normal - standard weight(i.e. 400)
  • bold - bold weight(i.e. 700)
  • bolder - bolder than the inherited font weight
  • lighter - lighter than the inherited font weight
  • 100(lightest)-900(boldest) - in multiples of 100

text-transform property

Description

The text-transform property controls the capitalization of text.

Usage

CSS

element {
  text-transform: value;
}

where value equals:

  • none - no capitalization, text renders as it is(default).
  • capitalize - transform the first character of each word to uppercase.
  • uppercase - transform all characters to uppercase.
  • lowercase - transform all characters to lowercase.
  • inherit - inherits this property from its parent element.

line-height property

Description

The line-height property specifies the line height.

Usage

CSS

element {
  line-height: value;
}

where value equals:

  • normal - normal line height(default).
  • number - a number that will be multiplied with the current font size to set the line height.
  • length - a fixed line height in px, em, etc.

Note: The number value is the preferred way to set line-height instead of length. There are no unexpected results in case of inheritance when using unitless numbers.

Example

HTML

<div class="green-box">
  <h3>Name: Marios Sofokleous</h3>
  <h3>Date: Mar 17, 2017</h3>
  <p>This is line number 1</p>
  <p>This is line number 2</p>
</div>

<div class="red-box">
  <h3>Name: Marios Sofokleous</h3>
  <h3>Date: Mar 17, 2017</h3>
  <p>This is line number 1</p>
  <p>This is line number 2</p>
</div>

CSS

/* Unitless number(preferred method) */
.green-box {
  line-height: 1.5;
  border: solid green;
  display: inline-block;
  vertical-align: top;
}

.red-box {
  line-height: 20px;
  border: solid red;
  display: inline-block;
  vertical-align: top;
}
