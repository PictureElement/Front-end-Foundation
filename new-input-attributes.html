New Input Attributes

PictureElement
Feb 28, 2017

HTML5 added the following attributes for the <input> element:

  • autocomplete
  • autofocus
  • form
  • formaction
  • formenctype
  • formmethod
  • formnovalidate
  • formtarget
  • height
  • width
  • list
  • min
  • max
  • multiple
  • pattern
  • placeholder
  • required
  • step

HTML5 added the following attributes for the <form> element:

  • autocomplete
  • novalidate

Placeholder

Description

The "placeholder" attribute specifies a hint that describes the expected value of an input field. The hint is shown inside the input field and hidden when the user starts typing.

Example

HTML5
<input type="text" name="first-name" placeholder="First name">
<input type="text" name="last-name" placeholder="Last name">
<input type="text" name="mail" placeholder="E-mail">
Result

Autofocus

Description

The "autofocus" attribute specifies that the input field should automatically get focus(active cursor) when the page loads.

Example

HTML5
<form>
  Username:<input type="text" name="username" autofocus>
  Password:<input type="password" name=:"psw">
</form>
Result
Username: Password:

Required

Description

The "required" attribute specifies that an input field must be filled out before submitting the form. It works with the following input types: "text", "search", "url", "tel", "email", "password", "date pickers", "number", "checkbox", "radio" and "file".

Example

HTML5
<form>
  Username: <input type="text" name="username" required>
  <input type="submit">
  Password: <input type="password" name="userpsw" required>
  <input type="submit">
</form>
Result
Username: Password:

Pattern

Description

Accepts a JS regular expression that can be used to validate a form field to match the pattern.

Example #1

HTML5
<form>
  Country code:
    <input type="text" name="two-letter_country_code" pattern="[A-Za-z]{2}"
    title="Two-letter country code">
  Telephone:
    <input type="tel" name="tel_number" pattern="[0-9]{10}" title="10-digit tel. number">
</form>
Result
Country code: Telephone:

When the form is submitted and the user's entry doesn't match the pattern, the user gets notified with an error message(i.e. "Please match the requested format").
