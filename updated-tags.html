"i" tag

Description

Represents text in an "alternate voice", mood, foreign words, technical terms or fictional character thoughts. It is a good idea to use the "class" attribute to make "i" more semantic.

Examples

  • I hope this works, he thought.
  • We ate different types of sushi last night such as Nigiri, Sashimi and Maki.
  • There are four variations of the Fox brevity word, Fox One is one of them and indicates a simulated or actual launch of a semi-active radar-guided missile.

"b" tag

Description

Represents "stylistically offset" text such as keywords and text that is bold by typographic convention(and not because is important). Also, the "b" tag can be used when you want to draw attention without increasing importance. It is a good practice to use the "class" attribute on the "b" tag in order to convey additional semantic information(i.e. <b class="lead"> for the first sentence in a paragraph). The "b" tag doesn't convey semantic information; use it only when no others fit.Do not mark titles and headings using the "b" tag.

Examples

  • The event takes place this upcoming Saturday, and over 3,000 people have already registered.

"em" tag

Description

Stress emphasis. If spoken, this stress would be emphasised pronunciation on a word.

Examples

  • Call a doctor now!
  • I love swimming

"strong" tag

Description

Strong importance.

Examples

  • Warning! This is very dangerous.
